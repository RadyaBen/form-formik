## Getting Started

### Prerequisites

* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org/en/download/)
* [npm](http://npmjs.com)

### Installation

```sh
# Step 1: Clone the repository
$ git clone https://github.com/RadyaBen/form-formik.git

# Step 2: Go into the repository 
$ cd form-formik

# Step 3: Install NPM packages
$ npm install

# Step 4: Run the app
$ npm start
```
